# Popular Go modules used as the reference corpus for typosquat detection.
# One module path per line; comments start with #. Major-version suffixes
# (/v2, and .v2 on gopkg.in) are stripped at load time.
#
# A dependency that IS one of these paths is always treated as clean.
# A dependency that is a near-miss of one of these paths is warned about.
#
# Inclusion criteria: the module must be widely depended upon (the kind of
# name an attacker would imitate) or a well-known legitimate project that
# shares a name with one. Pull requests adding such modules are welcome —
# each new entry reduces both missed squats and false positives.
